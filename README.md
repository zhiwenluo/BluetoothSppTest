BluetoothSppTest
================
用于ANDROID端蓝牙**SPP**服务,与仪器通信获取数据校验.

FEATURES
================
 - 能搜索附近**蓝牙设备**并请求配对
 - 以SPP服务进行配对,成功后发送帧数据进行通信
 - 有**建帧**相关函数(帧头,数据位,CRC校验码)
 - 有**解析帧**取出仪器数据的函数
 - 能设置连接时要输入的PIN，以自动跳过输入PIN步骤
